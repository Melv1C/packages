`@melv1c/rich-text-editor` is a composable rich text editor built on [Tiptap](https://tiptap.dev), with StarterKit enabled by default and UI primitives from `@melv1c/ui-core`.

## Features

- **Composable API** — Build your editor with `RichTextEditor`, `RichTextEditorToolbar`, and `RichTextEditorContent`.
- **Built-in presets** — Use `RichTextEditorBuiltIn` for a fast setup.
- **StarterKit by default** — Paragraphs, headings, lists, marks, history, and more are ready out of the box.
- **Customizable extensions** — Add your own Tiptap extensions through the `extensions` prop.
- **Typed primitives** — Full TypeScript support for editor props and toolbar components.
