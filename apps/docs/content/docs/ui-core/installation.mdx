## Requirements

- **React** 19 or later
- **Tailwind CSS** v4 or later
- **tw-animate-css** v1.4 or later

## Install the Package

```package-install
npm i @melv1c/ui-core
```

## Install Peer Dependencies

@melv1c/ui-core requires Tailwind CSS v4 and tw-animate-css as peer dependencies:

```package-install
npm i tailwindcss@^4 tw-animate-css@^1.4
```

## Configure CSS

Import the base styles in your main CSS file. You need to import the base CSS **and** a theme:

```css title="app.css"
@import 'tailwindcss';
@import 'tw-animate-css';
@import '@melv1c/ui-core/base.css';
@import '@melv1c/ui-core/themes/default.css';
```

### Available Themes

| Theme   | Import                               |
| ------- | ------------------------------------ |
| Default | `@melv1c/ui-core/themes/default.css` |
| Claude  | `@melv1c/ui-core/themes/claude.css`  |
| Vercel  | `@melv1c/ui-core/themes/vercel.css`  |
| Twitter | `@melv1c/ui-core/themes/twitter.css` |
| T3Chat  | `@melv1c/ui-core/themes/t3chat.css`  |
| Doom64  | `@melv1c/ui-core/themes/doom64.css`  |

See the [Theming guide](/docs/ui-core/theming) for more details on customizing themes.

## Set Up the Provider

Wrap your application with `UIKitProvider` to enable i18n and toast notifications:

```tsx title="app.tsx"
import { UIKitProvider } from '@melv1c/ui-core';

export function App({ children }: { children: React.ReactNode }) {
  return (
    <UIKitProvider i18nConfig="en" hasToaster>
      {children}
    </UIKitProvider>
  );
}
```

See the [Providers guide](/docs/ui-core/providers) for all configuration options.

## Verify Installation

Try importing a component to verify everything is set up correctly:

```tsx title="test.tsx"
import { Button } from '@melv1c/ui-core';

export function TestButton() {
  return (
    <Button variant="default" size="default">
      It works!
    </Button>
  );
}
```

If you see a styled button, you're all set!
