@melv1c/ui-core provides 44+ accessible, customizable React components. All components are built on [Radix UI](https://radix-ui.com) primitives and styled with [Tailwind CSS](https://tailwindcss.com).

## Import

All components can be imported from the main package:

```tsx
import { Button, Card, Dialog } from '@melv1c/ui-core';
```
