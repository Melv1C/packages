## Usage

```tsx
import { Checkbox } from '@melv1c/ui-core';
```

```tsx
<Checkbox id="terms" />
```

### Disabled

Use the `disabled` prop and `data-disabled` on the `Field` to disable a checkbox.

<ComponentPreview name="checkbox-disabled" />

### Group

Use `FieldSet` and `FieldLegend` to group related checkboxes.

<ComponentPreview name="checkbox-group" />

### Invalid

Use `data-invalid` on `Field` and `aria-invalid` on `Checkbox` to show an error state.

<ComponentPreview name="checkbox-invalid" />
