## Usage

```tsx
import { Button } from '@melv1c/ui-core';
import { toast } from 'sonner';
```

```tsx
<Button variant="outline" onClick={() => toast('Event has been created.')}>
  Show Toast
</Button>
```

> The `Toaster` is provided globally by `UICoreProvider`. No need to add it manually.
