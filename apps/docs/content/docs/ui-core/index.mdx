**@melv1c/ui-core** is the foundational package of the @melv1c packages monorepo. It provides 50+ production-ready, accessible React components based on [shadcn/ui](https://ui.shadcn.com) — built on top of [Radix UI](https://radix-ui.com) primitives and styled with [Tailwind CSS v4](https://tailwindcss.com) — and extended with custom higher-level components, blocks, and icons.

## Features

- **50+ Components** — Based on shadcn/ui and extended with custom components, blocks, and icons.
- **Multiple Themes** — Ships with 6 built-in themes: Default, Claude, Vercel, Twitter, T3Chat, and Doom64.
- **i18n Support** — Components with text content support 6 languages out of the box.
- **TypeScript** — Fully typed with TypeScript for excellent DX.
- **Tree-shakeable** — ESM-first, import only what you need.
