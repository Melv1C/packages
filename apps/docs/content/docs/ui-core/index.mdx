**@melv1c/ui-core** is the foundational package of the @melv1c packages monorepo. It provides 44+ production-ready, accessible React components built on top of [Radix UI](https://radix-ui.com) primitives and styled with [Tailwind CSS v4](https://tailwindcss.com).

## Features

- **44+ Components** — From basic elements like Button and Input to complex patterns like Sidebar, Command, and DateTimePicker.
- **Radix UI Primitives** — Built on battle-tested, accessible primitives.
- **Tailwind CSS v4** — Styled with utility-first CSS and fully customizable via CSS variables.
- **Multiple Themes** — Ships with 6 built-in themes: Default, Claude, Vercel, Twitter, T3Chat, and Doom64.
- **i18n Support** — Components with text content support 6 languages out of the box.
- **TypeScript** — Fully typed with TypeScript for excellent DX.
- **Tree-shakeable** — ESM-first, import only what you need.

## What's Included

### Base Components

Accordion, Alert, AlertDialog, Avatar, Badge, Breadcrumb, Button, ButtonGroup, Calendar, Card, Carousel, Checkbox, Collapsible, Command, ContextMenu, Dialog, Drawer, DropdownMenu, Form, HoverCard, Input, Label, Menubar, NavigationMenu, Pagination, Popover, Progress, RadioGroup, ScrollArea, Select, Separator, Sheet, Sidebar, Skeleton, Slider, Sonner (Toaster), Switch, Table, Tabs, Textarea, Toggle, ToggleGroup, Tooltip.

### Block Components

Pre-built, ready-to-use UI blocks like `LoginForm` with full validation, social auth providers, and i18n support.

### Custom Components

Higher-level components like `DateTimePicker`, `NextButton`, and `PreviousButton` that combine multiple primitives.

### Icons

Branded icon components for Apple, Facebook, GitHub, Google, and Microsoft with CVA variants.

### Providers

`UIKitProvider` — A root provider that wraps i18n and Toaster configuration for your entire app.
