@melv1c/ui-core uses CSS custom properties (variables) for theming, making it easy to switch between pre-built themes or create your own.

## Built-in Themes

The package ships with 6 ready-to-use themes:

| Theme       | Description                      | Import                               |
| ----------- | -------------------------------- | ------------------------------------ |
| **Default** | Clean, neutral design            | `@melv1c/ui-core/themes/default.css` |
| **Claude**  | Warm, Anthropic-inspired palette | `@melv1c/ui-core/themes/claude.css`  |
| **Vercel**  | Sharp, monochrome design         | `@melv1c/ui-core/themes/vercel.css`  |
| **Twitter** | Blue-accented social theme       | `@melv1c/ui-core/themes/twitter.css` |
| **T3Chat**  | Modern chat-inspired theme       | `@melv1c/ui-core/themes/t3chat.css`  |
| **Doom64**  | Dark, gaming-inspired theme      | `@melv1c/ui-core/themes/doom64.css`  |

## Using a Theme

Import the base styles and your chosen theme in your CSS file:

```css title="app.css"
@import 'tailwindcss';
@import 'tw-animate-css';
@import '@melv1c/ui-core/base.css';

/* Pick one theme */
@import '@melv1c/ui-core/themes/default.css';
```

To switch themes, simply change the import:

```css title="app.css"
@import 'tailwindcss';
@import 'tw-animate-css';
@import '@melv1c/ui-core/base.css';

/* Using the Claude theme instead */
@import '@melv1c/ui-core/themes/claude.css';
```

## Dark Mode

All themes include both light and dark mode support. Dark mode is automatically applied based on the `dark` class on your `<html>` element:

```html
<!-- Light mode -->
<html>
  ...
</html>

<!-- Dark mode -->
<html class="dark">
  ...
</html>
```

## Overriding Specific Variables

You don't need to define every variable. Override only the ones you want to change:

```css title="app.css"
@import 'tailwindcss';
@import 'tw-animate-css';
@import '@melv1c/ui-core/base.css';
@import '@melv1c/ui-core/themes/default.css';

/* Override just the primary color */
:root {
  --primary: 262 83% 58%;
  --primary-foreground: 0 0% 100%;
}

.dark {
  --primary: 262 83% 68%;
  --primary-foreground: 0 0% 0%;
}
```

## Using Theme Variables in Your Code

You can reference theme variables in your Tailwind classes or custom CSS:

```tsx
// Using Tailwind classes
<div className="bg-primary text-primary-foreground">
  Themed content
</div>

// In custom CSS
.my-custom-element {
  background-color: hsl(var(--primary));
  color: hsl(var(--primary-foreground));
}
```
